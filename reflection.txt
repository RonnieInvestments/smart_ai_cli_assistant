5. Minimal Working Example

Python CLI sends request to local Ollama API

import requests

def ask_ai(messages):
    response = requests.post(
        'http://localhost:11434/api/chat',
        json={
            'model': 'tinyllama',
            'messages': messages,
            'stream': False
        }
    )
    return response.json()['message']['content']

messages = [{'role': 'user', 'content': 'What is AI?'}]
print(ask_ai(messages))

7. Challenges Faced

- Model loading failures due to insufficient RAM
- 500 Internal Server Error when using larger models
- Incorrect API payload format when calling Ollama
- Confusion between terminal chat mode and API mode

8. How Challenges Were Solved

- Switched to lightweight model (tinyllama)
- Ensured Ollama server was running in background
- Fixed API request format (/api/chat with messages structure)
- Added debugging prints to inspect server responses

9. Future Improvements

10. Common Issues & Fixes
